NGINX Dockerfile Lab

Chào mừng đến với lab Dockerfile cho NGINX!

Build Context và Layers

Dockerfile này được build với các layer sau:

  • Base Image: alpine:3.20 - Image nhẹ dựa trên Alpine Linux
  • Installation Layer: Cài đặt nginx, bash, và curl bằng apk package manager
  • Configuration Layer: Copy nginx.conf và default.conf vào container
  • Content Layer: ADD html.tar.gz được giải nén vào /usr/share/nginx/html/

Mỗi layer được cache lại để tối ưu hóa quá trình build lần sau. Khi thay đổi nội dung, chỉ layer bị ảnh hưởng mới được rebuild.

Container này chạy nginx ở foreground mode (daemon off) để có thể xem logs qua docker logs và quản lý process lifecycle tốt hơn.
